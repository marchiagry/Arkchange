交通總原則

  • 市區：Suica / PASMO 手機卡最方便，JR＋東京 Metro 混用即可。
  • 成田機場：若住新宿，優先 N'EX 直達；想省預算可搭京成 Skyliner 到日暮里再轉 JR。
  • 河口湖：新宿高速巴士最直覺，旺季/週末需先訂位；遇大雨直接改室內日。
  • 迪士尼：新宿 → 東京/八丁堀 → 舞濱，再搭迪士尼單軌到 DisneySea。

雨天備案使用法

  • 小雨：照原行程，但把戶外散步縮短，餐廳/百貨停留拉長。
  • 大雨強風：河口湖、明治神宮、代代木公園等戶外點優先替換。
  • 備用室內區：新宿地下街、銀座百貨、東京站一番街、池袋 Sunshine City、墨田水族館。

抵達東京 → 進新宿 → 晚餐與初採買

機場移動日低體力晚餐＋藥妝

成田 T1 約 35.7631, 140.3864；新宿站約 35.6903, 139.7005（OSM/Nominatim）。

雨天備案：全程改走新宿地下街、Lumine、NEWoMan、伊勢丹、京王/小田急百貨；晚餐挑車站連通或離出口 3–5 分鐘內餐廳。

DAY2

淺草 → 晴空塔 → 上野 → 秋葉原

東東京一線拍照＋散步上野阿美橫＋電器動漫

新宿→淺草寺/雷門→晴空塔→上野/阿美橫→秋葉原→新宿

DAY4

河口湖富士山一日：湖景、纜車、溫泉/咖啡

遠程日需早出門天氣決定成敗
建議交通動線
新宿高速巴士總站→河口湖站→天上山纜車→河口湖畔/大石公園→回新宿
河口湖站約 35.4985, 138.7686（OSM/Nominatim）。高速巴士旺季一定先預約，回程也一起訂。
交通打法新宿高速巴士直達河口湖最省腦；也可 JR 特急到大月轉富士急行，但轉乘較多。
停留策略天氣好先拍富士山，不要把湖景放到傍晚；午餐/咖啡作為彈性緩衝。
雨天備案：若預報大雨/雲厚看不到富士山，建議與 D3 或 D5 對調。當天改 新宿百貨＋池袋 Sunshine City/水族館＋無印良品/Loft 採買；河口湖不要硬衝，交通時間長但景色折損大。
新宿→河口湖河口湖站→纜車雨備：池袋 Sunshine City
DAY5

澀谷 → 原宿 → 表參道 → 新宿夜購

西東京潮流線年輕品牌購物主力日
建議交通動線
新宿→澀谷 Scramble / Shibuya Sky→原宿竹下通→明治神宮/表參道→新宿
交通打法新宿到澀谷搭 JR 山手線；澀谷→原宿同線一站，也可沿 Cat Street 慢慢走到表參道。
吃買重點澀谷 Parco/Loft/109、原宿小店、表參道甜點咖啡，晚上回新宿補百貨。
雨天備案：明治神宮與 Cat Street 縮短，改排 Shibuya Scramble Square、Hikarie、Parco、Omotesando Hills 室內購物線。大雨時不要走太長戶外路段。
新宿→澀谷澀谷→原宿原宿→表參道
DAY6

Tokyo DisneySea 全日

早出晚歸排隊體力日APP/票券先準備
建議交通動線
新宿→東京/八丁堀→舞濱→Disney Resort Line→DisneySea→新宿
Tokyo DisneySea 約 35.6273, 139.8844（OSM/Nominatim）。
交通打法預留 70–90 分鐘從新宿到園區入口；回程人潮大，別把隔天早班機排太緊。
園區策略入園先處理熱門設施/付費項目，午餐錯峰，傍晚看體力決定是否留到夜間秀。
雨天備案：小雨 DisneySea 仍可玩，但準備輕便雨衣、防水鞋套、行動電源防水袋。若遇強風豪雨且可調票，改成 台場 DiverCity/Aqua City＋豐洲/有明室內購物，保留迪士尼到天氣較穩日。
新宿→DisneySea雨備：台場 DiverCity
DAY7

新宿最後採買 → 機場回程

收尾日低風險伴手禮/藥妝補貨
建議交通動線
飯店退房寄放行李→新宿最後採買/午餐→取行李→N'EX / Limousine Bus / Skyliner→成田機場
交通打法回程日不要跨遠區；以新宿站周邊完成所有補貨，至少抓國際線起飛前 3 小時抵達機場。

新宿→成田機場新宿伴手禮地圖

行前檢查清單

  • Suica/PASMO 手機卡或實體交通卡。
  • 河口湖高速巴士來回先訂，遇壞天氣再評估改期。
  • DisneySea 門票、官方 APP、行動電源。
  • 雨衣/折傘、防水鞋、塑膠袋或防水袋。

小露提醒

最需要保留彈性的兩天：D4 河口湖、D6 DisneySea。東京市區行程室內備案很多，真正該看天氣調整的是富士山與遊樂園。
購物節奏：前半段先看價與品項，D5/D7 再集中買；避免每天提重物走太多路。
